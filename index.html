    Keyboard Shortcuts

    • Enter: Add Task (when input focused)
    • Ctrl+N: Add Task
    • /: Focus Task Input
    • Arrow Up/Down: Navigate Tasks
    • Ctrl+Enter: Complete Selected Task
    • Delete: Delete Selected Task
    • Ctrl+S: Sort Tasks
    • Ctrl+Shift+C: Clear All Tasks
    • ?: Show Shortcut Help
    • Esc: Close Shortcut Help
